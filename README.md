# ITI CG TD Track | Intake 46 - Group 2

Welcome to the official repository for **Group 2** of the **Computer Graphics Technical Director (CG TD)** track, Intake 46, at the **Information Technology Institute (ITI)**. This repository serves as a collaborative hub for our pipeline tools, automation scripts, and technical solutions developed during our training.

## üöÄ Track Overview
The CG Technical Director track focuses on bridging the gap between artistic vision and technical implementation, including the following roles:
*   **Pipeline Development:** Building robust workflows for production.
*   **Tool Development:** Creating custom UIs and tools using Python & PySide/PyQt.
*   **DCC Scripting:** Automating tasks in Maya (Maya Python API/cmds) and Houdini (HOM).
*   **Math for CG:** Technical implementations of geometry, vectors, and physics.
*   **Shading & Rendering:** Automation and optimization for render engines.
